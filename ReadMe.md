# ⚛️ React Web Development (DEPI Portfolio)

Welcome to the **React Web Development** project hub completed under the **Digital Egypt Pioneers Initiative (DEPI)**. This repository hosts a variety of training materials, assignments, JavaScript challenges, and interactive React/JavaScript applications.

---

## 🛠️ Technology Stack & Focus Areas

This training track covers modern web development paradigms:
- **Core Frontend:** Semantic HTML5 and responsive CSS3.
- **JavaScript Essentials:** Advanced ES6+, OOP, asynchronous operations, array methods, DOM manipulation, and CRUD architectures.
- **React.js Development:** Component-driven design, state management, props, hooks, API integration, and routing.
- **Responsive Web Design:** CSS Flexbox, Grid, custom styling, and media queries.

## ⚙️ Running the Projects

### Running React Applications (e.g., `movies-app`)
1. Ensure Node.js is installed.
2. Navigate to the project directory:
   ```bash
   cd movies-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev  # or npm start
   ```

### Running Static/Vanilla JS Projects
1. Double-click the `index.html` file in folders like `CRUDS/`, `Gallary-Slider/`, or the landing pages to run them directly in your browser.
2. Alternatively, run them via a local server extension in your IDE or Python CLI:
   ```bash
   python3 -m http.server 8000
   ```
